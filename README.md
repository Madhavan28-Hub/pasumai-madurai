### 📋 Project Overview

**Pasumai Madurai** is an AI-powered waste segregation assistant built for the **1M1B Virtual Internship** in collaboration with **IBM SkillsBuild** and **AICTE**. This web application helps households and local communities in Madurai properly sort their household waste to prevent landfill pollution, directly supporting **UN Sustainable Development Goals 11** (Sustainable Cities and Communities) and **12** (Responsible Consumption and Production).

### 🎯 Mission

To guide Madurai residents on how to cleanly sort household waste into 4 municipal streams:
- 🟢 **Green Bin** (மட்கும் குப்பை) - Biodegradable Waste
- 🔵 **Blue Bin** (மட்காத குப்பை) - Non-biodegradable Recyclable
- 🔴 **Red Bin** (ஆபத்தான கழிவுகள்) - Domestic Hazardous Waste
- ⚡ **E-Waste** (மின்னணுக் கழிவுகள்) - Electronic Scraps

### 🚀 Getting Started

#### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs directly in browser

#### Usage
1. Open `index.html` in your web browser
2. Type any household item in the chat input (e.g., "plastic bottle", "banana peel")
3. Receive instant guidance on proper disposal
4. Or click example chips for quick demonstrations

#### Example Queries
```
✅ "banana peel" → Green Bin guidance
✅ "plastic bottle" → Blue Bin with recycling tips
✅ "old mobile" → E-waste disposal instructions
✅ "medicine strip" → Hazardous waste handling
✅ "வாழைப்பழ தோல்" → Works in Tamil too!
```

### 📊 Waste Categories Covered

#### 🟢 Green Bin (Biodegradable)
- Food waste (fruits, vegetables, cooked food)
- Garden waste (leaves, grass, flowers)
- Organic materials (paper, cardboard, wood)

#### 🔵 Blue Bin (Recyclable)
- Plastics (bottles, containers, bags)
- Metals (cans, foil, utensils)
- Glass (bottles, jars)
- Paper products (newspapers, magazines)
- Textiles (old clothes, cloth bags)

#### 🔴 Red Bin (Hazardous)
- Medical waste (medicines, syringes, bandages)
- Sanitary waste (pads, diapers)
- Chemical waste (batteries, paint, pesticides)

#### ⚡ E-Waste (Electronics)
- Mobile devices (phones, laptops, tablets)
- Home appliances (TV, refrigerator, fan)
- Electronic accessories (chargers, cables, bulbs)
- Batteries (all types)
